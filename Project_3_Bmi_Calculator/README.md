# Bmi Calculator

The Bmi is Calculatoed by dividing a person's weight (in kg) by the square of their height (in m): 

![](https://i.imgur.com/VZDEcu9.png)

## Example input
```bash
weight = 85
height = 1.75
```

## Example output
```bash
Your BMI is 28, you are slightly overweight.
```
