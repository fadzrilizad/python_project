# Treasure Map

- Program that allows mark square using two-digit system
- First digit is vertical and second is horizontal

![](https://i.imgur.com/JxhijM4.png)

## Example input

- Column 2, row 3 would be entered as:

```bash
23
```

## Example output

```bash
['⬜️', '⬜️', '⬜️']
['⬜️', '⬜️', '⬜️']
['⬜️', 'X', '⬜️']
```
