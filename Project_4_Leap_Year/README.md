# Leap Year

Normal Year  has 365 days, leaps years has 366, extra days in february

## Codition Leap Year
- Every year divisible by 4 is Leap Year
- Except is evenly divisible by 100
- Unless the year is also divisible by 400

# Example input 1
```bash
2400
```

# Example output 1
```bash
Leap Year
```

# Example input 2
```bash
1988
```

# Example output 2
```bash
Not leap Year
```
